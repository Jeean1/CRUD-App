



CRUD App


Una aplicación dónde puedes realizar operaciones CRUD.

Tiene un lado visual & un server, hechos en React & Node JS.


Cómo instalar:

  Ejecutamos en la carpeta principal:
        -npm i
        -npm run install:app  (Instala librerias & modulos)
        -npm run start:app (Enciende el servidor y activa la parte visual)


    A continuación ya se puede utilizar la aplicación.

    Link de Documentación: https://documenter.getpostman.com/view/22441762/2s8YzNzids
